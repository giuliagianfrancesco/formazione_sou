String Sorter & Duplicate Remover
A Bash script designed to normalize, sort, and deduplicate strings of text.

Features
Case-Insensitive Conversion: Converts all input to lowercase to ensure consistent comparisons.

Alphabetical Sorting: Implements a Bubble Sort algorithm to organize words.

Deduplication: Filters out repeated terms to return a list of unique values.


Requirements
Unix-like environment (Linux, macOS, WSL).
